# CONTRIBUTING
## Introduction
Thank you for considering contributing to OmniPublish-Multi-Platform-Content-Distribution-Node-CLI-Tool! We are excited to have you on board and value your input.
## Getting Started
1. **Fork the repository**: https://github.com/chirag127/OmniPublish-Multi-Platform-Content-Distribution-Node-CLI-Tool/fork
2. **Clone the repository**: `git clone https://github.com/your-username/OmniPublish-Multi-Platform-Content-Distribution-Node-CLI-Tool.git`
3. **Install dependencies**: `npm install`
4. **Build and test**: `npm run build` and `npm run test`
## Contribution Guidelines
* **Code quality**: Follow standard professional guidelines for TypeScript and JavaScript.
* **Testing**: Ensure all changes are covered by tests.
* **Documentation**: Update README.md and other relevant documentation as needed.
* **Commit messages**: Follow the conventional commit message format.
## Submitting a Pull Request
1. **Create a new branch**: `git checkout -b your-branch-name`
2. **Make your changes**: Edit files, add new ones, etc.
3. **Commit your changes**: `git add .` and `git commit -m "your-commit-message"`
4. **Push your changes**: `git push origin your-branch-name`
5. **Create a pull request**: https://github.com/chirag127/OmniPublish-Multi-Platform-Content-Distribution-Node-CLI-Tool/pull/new/your-branch-name
## Code of Conduct
* Be respectful and professional in all interactions.
* Follow the GitHub Community Guidelines: https://docs.github.com/en/site-policy/github-terms/github-community-guidelines
## Reporting Issues
* **Create an issue**: https://github.com/chirag127/OmniPublish-Multi-Platform-Content-Distribution-Node-CLI-Tool/issues/new
* **Provide details**: Describe the issue, including steps to reproduce and expected behavior.
