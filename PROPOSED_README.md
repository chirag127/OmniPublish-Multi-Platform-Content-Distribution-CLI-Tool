# OmniPublisher-MultiPlatform-Content-Syndicator

A powerful engine to automate multi-platform content publishing and static site archival from a single Markdown source. Syndicates posts across Dev.to, Medium, Hashnode, WordPress, LinkedIn, Reddit & more, ensuring idempotency and seamless CI/CD integration.

## üéØ Overview

OmniPublisher is a sophisticated TypeScript-based engine designed to streamline content creation workflows. It empowers users to manage their content in a single Markdown source and automate its syndication to a wide array of platforms, including Dev.to, Medium, Hashnode, WordPress, LinkedIn, and Reddit. Simultaneously, it archives content as static sites, ensuring a robust and future-proof publishing strategy.

---

## üìà Key Features

*   **Single Source of Truth:** Manage all content within Markdown files.
*   **Multi-Platform Syndication:** Publish seamlessly to Dev.to, Medium, Hashnode, WordPress, LinkedIn, Reddit, and more.
*   **Idempotent Publishing:** Ensure content is not duplicated or overwritten unintentionally.
*   **Static Site Archival:** Generate static archives of published content.
*   **CI/CD Integration:** Designed for robust integration into automated build and deployment pipelines.
*   **Extensible Platform Adapters:** Easily add support for new publishing platforms.
*   **Configuration Management:** Flexible configuration options for API keys, platform endpoints, and publishing rules.

## üîß Getting Started

### Prerequisites

*   Node.js (v20.x or higher recommended)
*   npm or Yarn package manager

### Installation

1.  **Clone the repository:**
    bash
    git clone https://github.com/chirag127/OmniPublisher-MultiPlatform-Content-Syndicator.git
    cd OmniPublisher-MultiPlatform-Content-Syndicator
    

2.  **Install dependencies:**
    bash
    npm install
    # or
    yarn install
    

### Configuration

Configuration details, including API keys and platform-specific settings, should be managed via environment variables or a dedicated configuration file (e.g., `.env`, `config.json`). Refer to the [Configuration Guide](https://github.com/chirag127/OmniPublisher-MultiPlatform-Content-Syndicator/blob/main/docs/configuration.md) for detailed instructions.
